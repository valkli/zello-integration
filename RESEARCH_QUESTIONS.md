## Конкретные технические детали для проверки

1. **Структура бинарного пакета:**
   ```
   Входящий пакет: 01000029330000000018c1...
   - 01 = packet type
   - 00002933 = stream_id (Big Endian) = 10547
   - 00000000 = packet_id (Big Endian) = 0
   - 18c1... = Opus payload (начинается с TOC byte 0x18)
   
   Исходящий пакет: должен быть таким же?
   ```

2. **Codec header:**
   ```
   Входящий: gD4BPA== (base64) = 803e013c (hex)
   - 803e = 16000 Hz (Little Endian)
   - 01 = frames per packet
   - 3c = 60ms frame duration
   
   Исходящий: используем такой же формат?
   ```

3. **Opus TOC byte:**
   ```
   Входящий: 0x18 = 00011000
   - Config: 0 (bits 0-2)
   - Stereo: 1 (bit 3)
   - Frame count: 1 (bits 4-6)
   
   Исходящий: должен быть таким же?
   ```
